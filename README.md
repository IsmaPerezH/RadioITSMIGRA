# ¡Bienvenido a Radio ITSMIGRA (App Expo) 

## Descripción del proyecto
RadioITSMIGRA es una aplicación móvil desarrollada con **Expo** y **React Native** que permite a los usuarios escuchar en tiempo real la emisora **Radio ITSMIGRA** (96.5 FM) de San Miguel el Grande, Oaxaca. La app muestra información de la estación (nombre, frecuencia, ubicación, slogan, logo, teléfono y sitio web) y ofrece un reproductor de streaming integrado. Utiliza un tema oscuro con los colores institucionales de la radio (azul primario y rojo secundario) y está estructurada con componentes reutilizables para la información de la estación y el reproductor.
